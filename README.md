# React + Vite

This project is currently built for Aksamedia internship recruitment test.

## How to run locally

1. Clone the repo

```bash
git clone https://github.com/Luthfiyanto/Aksamedia-test.git
```

2. Install dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```
